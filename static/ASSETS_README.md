# Assets Required

This document lists the image assets needed for the StarkHacks website.

## Navigation Assets

- **Logo Icon**: `/static/logo-icon.png` (70x69px)
  - The robot head/face icon (yellow/orange with blue details)
  - This is the visual symbol part of the logo
- **Logo Text**: `/static/logo-text.png` (128x42px)
  - The text "Humanoid Robot Club"
  - "Humanoid" on first line, "Robot Club" on second line
  - White text on transparent background

## Hero Section Assets

- **Hero Background**: `/static/hero-background.jpg` - Full-width background image for hero section
  - Recommended size: 1920x1080px or larger
  - Format: JPG (optimized) or WebP for better performance

## Image Optimization Guidelines

1. Use WebP format when possible for better compression
2. Provide multiple sizes for responsive images
3. Optimize images before adding to static folder
4. Use lazy loading for images below the fold

## Placeholder Images

Until actual assets are provided, the components use CSS-based placeholders that match the design dimensions.
